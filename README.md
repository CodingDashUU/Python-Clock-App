# Python-Clock-App
A simple clock to display your current time in 24 hour format, also marks the very first desktop app I made for Windows.

## Technologies Used:
1. Python
2. PyQt5

## Modules Used
1. datetime
2. time
3. PyQt5.QtWidgets
  1. QApplication
  2. QMainWindow
  3. QLabel
  4. QWidget
  5. QVBoxLayout
4. sys
5. PyQt5.QtCore
   1. Qt
   2. QTimer
  
## How to use
1. Download the clock.exe file in releases
2. Double Click the exe file to be able to use the application
